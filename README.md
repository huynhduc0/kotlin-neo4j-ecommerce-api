# eshop_api
Project on going construct

A API backend project using Spring Boot with kotlin, jpa
I will add recommend system, bad comment detection soon
Please install jdk and run with maven
# Some Info:
- support 4 role: ADMIN,MERCHANT,USER, SYSTEM
- Products are come from merchants and buy with user has role USER
- Admin can be management any thing
- Merchants can be management their shop
- Merchants and admin can view review with bad/good comment auto detect by system
- Payment with stripe
